Description
-----------
 MorseCodeConverter lets anyone translate text to Morse code easily.
 It allows users to convert text into Morse code, providing an interactive way to explore  and work with Morse code translations.
 Morse code is a method for transmitting telegraphic information, using standardized sequences of short and long  elements to represent the letters, numerals, punctuation and special characters of a message. The short and long  elements can be formed by sounds, marks or pulses, and are commonly known as "dots" and "dashes" or "dits" and  "dahs".


Installation
------------
 1. Double-click "MorseCodeConverter.setup" to start the installation process
 2. Follow the on-screen instructions
 3. Wait for the installation to complete
 4. Launch the software

That's it! You have successfully installed MorseCodeConverter on your computer. I hope this guide was helpful, and feel free to ask if you have any questions or need further assistance.


Usage
--------
 Just type in the text input field and press convert to use the Morse code converter. 

 Examples:

 What is SOS in Morse Code?
 SOS in Morse code is "... --- ..."

 How Do You Say Hello in Morse Code?
 "Hello" in Morse code is ".... . .-.. .-.. ---"

 How Do You Say Help in Morse Code?
 "Help" in Morse code is ".... . .-.. .--.", but you can also use the SOS signal, which is "... --- ...".


Uninstallation
--------------
  Run Uninstall from 'MorseCodeConverter' folder in the Start menu or go to 
  Control Panel, launch "Add or Remove Program" applet and find the program 
  here.
  
  Please note, you do not need to unistall the program before installation of a 
  new version! Just install it over an old version.
